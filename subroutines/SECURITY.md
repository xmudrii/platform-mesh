# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/platform-mesh/subroutines/security) of this repository.
2. Click **"Report a vulnerability"**.
3. Provide a description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 5 business days and work with you to understand and address the issue.

## Supported Versions

As this project is in alpha stage, only the latest release is supported with security updates.

## Disclosure Policy

When we receive a security report, we will:

1. Confirm the vulnerability and determine its impact.
2. Prepare a fix and release it as soon as possible.
3. Credit the reporter (unless they prefer to remain anonymous).
